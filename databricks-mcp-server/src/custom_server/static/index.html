ðŸš€ Setup Instructions

1 Download MCP Client

Download the MCP client and save it to a folder on your computer:

C:\Users\YourName\mcp\databricks_mcp_client.js

2 Get Your Authentication Token

Visit the dashboard page to get your current JWT token.

Important: Tokens expire every hour. You'll need to refresh them periodically.

3 Configure Claude Desktop

Update your Claude Desktop configuration file:

Windows: %APPDATA%\Claude\claude_desktop_config.json

Mac: ~/Library/Application Support/Claude/claude_desktop_config.json

{
  "mcpServers": {
    "unity-catalog": {
      "command": "node",
      "args": ["C:\\path\\to\\your\\databricks_mcp_client.js"],
      "env": {
        "BEARER_TOKEN": "your-jwt-token-from-step-2"
      }
    }
  }
}

4 Restart Claude Desktop

Close and reopen Claude Desktop to load the new MCP server configuration.
